# Version 2.4 — Scoring and Goalie Timing Reminders

- Adds a dedicated, high-visibility transition card after each period's shots.
- Shows the updated running score after every goal.
- Highlights empty-net goals and calculates the prior whole-minute goalie pull time.
- Reminds the scorer to return the goalie at the exact empty-net goal time.
- Detects and highlights penalty-shot goals, with matching goalie Off and Back On times.
- Adds regression tests for period transitions, running scores, empty-net timing, and penalty-shot timing.
